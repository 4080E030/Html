# 選擇器可以定義某組 CSS 樣式要套用到哪些元素上。
## Universal Selector
```html
* {
  color: green;
}
米字號可以讓全部的網站都變成你要變成的樣式
例如我設定成 color:green; 就代表說我網站內的文字都會變成綠色
```
