# rgb
```html
這個就是我們使用的rgb程式碼

h1 {
  color: rgb(0, 0, 0);
}
(0,0,0) 代表得是red,green,blue 
三個數字都是介於0~255種 如果都是0，代表就是黑色
如果是(255,0,0)就是紅色以此類推 
```
